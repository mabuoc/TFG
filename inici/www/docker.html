Configuració de Docker

Primer actualitzem els repositoris i el sistema amb les ordres:

        
            sudo apt update
            sudo apt dist-upgrade

Instal·lem el paquet docker.io:

        
            sudo apt install -y docker.io

Activem docker a l’arrencada del sistema i forcem a que s’iniciï ara mateix:

        
            sudo systemctl enable docker --now

A partir d’aquest moment ja tenim docker instal·lat. Per tal de que el nostre usuari pugui utilitzar docker, l’hem d’afegir al grup docker:

        
            sudo usermod -aG docker kali
